## AP Calculus Question Selector

This repository contains AP Calculus test banks (TB_1 through TB_7) and a question selector tool that allows you to filter questions by College Board AP Calculus AB or BC chapters.

### Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the question selector:
```bash
python3 question_selector_cli.py
```

For detailed usage instructions, see [USAGE.md](USAGE.md).
